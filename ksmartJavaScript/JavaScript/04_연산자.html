연산자

산술연산자

- +, -, *, /, %

문자결합연산자

- '문자' + 1 , 1 + '문자' => 문자열 연결

대입 연산자 및 복합 대입 연산자

- 일반적인 산술연산자를 간소화시켜 쓰는 방식
- =, +=, -=, *=, /=, %=

증감연산자

- ++, --

비교연산자

- 두개의 데이터를 비교하여 결괏값을 논리형으로 받는다. - ==(값만 비교), ===(값과 타입모두 비교), !=, !==

논리연산자

- 2개 이상의 결괏값 혹은 비교연산자를 동시에 비교
- and(&&) -> 2개 이상의 비교값이 모두 참일 경우 참
- or(||) -> 2개 이상 중 하나라도 참이면 참
- not(!) -> 결괏값의 반대

삼항조건연산자

- 조건식을 비교하여 참일 경우와 거짓일 경우 판별하여 각각의 값을 반환

null 병합연산자

-구문 변수(값)??결과값 -> 변수의 값이 null 일 경우 우항의 값으로 반환
- 변수(값)이 null이 아닌 경우는 변수에 담겨져있는 값 유지
- ex) null? 'null일 경우 반환값';
