if, else if, else

- if 안에 있는 조건식이 참인 경우 해당하는 if문, else if문을 실행 한다. - 조건문(if, else if)에 모두 해당하지 않으면 else에 있는 statement3이 실행된다. - else if에는 갯수 제한이 따로 없다.

switch

- 값 1개를 여러번 동시에 비교(같은 값) 할 수 있는 문장
- case 문에 삽입된 값과 변수값 일치 할 경우 : 다음 에 오는 코드 실행
- break; -> switch 문 중단
- default; -> case의 값과 일치한 내용이 없을 경우
